# Demo marker

`gastroar-card-v1.png` and `gastroar-card-v1.mind` come from the MindAR 1.2.5 image-tracking card example (MIT). They are the default development target.

Print the target image at exactly 90 mm wide. The printable restaurant route places this reference beside a generated table QR. Restaurants can upload their own preview and matching compiled `.mind` target from the admin console.
